Telugu Sandhi Rules — Simple Explanation

Sandhi is the sound change that happens when two letters or words join.

1. Savarna Deergha Sandhi

అ + అ → ఆ

Example: రామ + అలయం → రామాలయం

2. Guna Sandhi

అ + ఇ → ఏ
అ + ఉ → ఓ

Example: రాజ + ఇంద్రుడు → రాజేంద్రుడు

3. Vriddhi Sandhi

అ + ఏ → ఐ
అ + ఓ → ఔ

Example: దేవ + ఏశ్వరుడు → దేవైశ్వరుడు

4. Hallu Sandhi

Example: సత్ + గుణం → సద్గుణం

5. Visarga Sandhi

Example: గురుః + ఆగచ్ఛతి → గురురాగచ్ఛతి
